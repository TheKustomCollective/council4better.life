# Council For A Better Life

A community platform to make your voice heard, for council engagement and better living for everyone no matter socio-economic status. This platform enables citizens to connect with their local council, stay informed about decisions, and participate in community discussions.Further if there is need unmet, an opportunity to help others the Council is there.

## Getting Started

### Prerequisites

- Node.js 18+ and npm

### Installation

1. Clone the repository:
```bash
git clone https://github.com/TheKustomCollective/council4better.life.git
cd council4better.life
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:3000`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
- `npm run format` - Format code with Prettier

## Benefits Resolution System

The Benefits Resolution System is a dedicated development area for building tools to help citizens access benefits and support services. This feature aims to bridge gaps in socio-economic access to essential services.

### Current Features:
- **Application Management**: Track and manage benefit requests with urgency levels
- **Resource Database**: Comprehensive information about available benefits and how to apply
- **Anonymous Submissions**: Privacy-protected request system
- **Status Tracking**: Monitor application progress from submission to resolution

### Development Areas:
The system includes clearly marked development placeholders for:
- Case notes and timeline tracking
- Applicant communication system
- Document upload/management
- Eligibility pre-screening
- AI-powered benefit matching
- Real-time validation and assistance
- SMS/Email notification preferences
- Integration with external benefit databases

This is your workspace to expand and complete the Benefits Resolution System according to community needs.
