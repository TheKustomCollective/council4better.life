# Council for Better Life

Welcome to the Council for Better Life project repository. This project provides resources and services for end-of-life planning and legacy preservation.

## Living Legacy

The [LivingLegacy](./LivingLegacy) project is our comprehensive media kit and awareness campaign for end-of-life concierge services, helping individuals and families navigate important life transitions with dignity and support.

### Key Features
- Complete Media Kit for Living Legacy end-of-life concierge services
- Comprehensive documentation on services and benefits
- Press resources and media outreach templates
- Client testimonials and success stories

## About This Repository

This repository was imported from the commerce2 project to establish a dedicated home for the Council for Better Life initiatives, focusing on:
- End-of-life planning support
- Legacy preservation services
- Educational resources for families
- Community awareness campaigns
